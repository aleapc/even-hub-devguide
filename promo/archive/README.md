# Archived promo materials

This folder holds the first round of community outreach drafts, written on 2026-04-11 to announce this repo as a "developer guide" for the Even Realities G2 platform.

**These are archived and will not be used.** A second outreach round will not happen.

## Why

The repo has pivoted from "competitive developer guide" to **case studies complementing the existing community references**:

- SDK reference & community documentation lives at [`nickustinov/even-g2-notes`](https://github.com/nickustinov/even-g2-notes).
- UI library / design system lives at [`fabioglimb/even-toolkit`](https://github.com/fabioglimb/even-toolkit).
- Official starters: [`even-realities/evenhub-templates`](https://github.com/even-realities/evenhub-templates).
- Claude Code plugin for G2: [`even-realities/everything-evenhub`](https://github.com/even-realities/everything-evenhub).

This repo focuses on **architectural patterns beyond the glasses themselves** (mobile companion, distributed backend, telemetry, pixel-art pipeline, testing without Jest) plus case studies of the 7 production apps. See the top-level [README](../../README.md) for the current framing.
